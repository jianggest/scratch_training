# scratch_training
A scratch training class for children.

课件的制作按照以下几个内容点：

【引入引爆】（5-10分钟）： 玩一玩今天要做的成品游戏，或者看一个生活现象。提问：你觉得这是怎么做到的？（激发兴趣）
【核心逻辑/现实比喻】（10-15分钟）： 脱离电脑！用画板、实物、比喻来讲解今天的核心概念（如变量、递归）。（建立计算思维）
【实战演练：核心原型】（20-30分钟）： 老师带着写最核心的几行代码。遇到问题不要直接给答案，而是带着他看报错、看运行轨迹。（代码实现）
【挑战升级：独立开发】（15-20分钟）： 给出1-3个进阶小任务（比如：改变颜色、增加分数、加音效），让学生自己动手试错。（巩固知识）
【复盘与展示】（5分钟）： 总结今天学到的“魔法”，回顾踩过的“坑”。
